Lichen Mapping

Rosemary Victoria Greensmith 2025-09-19

This project maps records of various lichen species which are indicators of gaseous nitrogen (N2). The records are from the National Biodiversity Network (NBN) Atlas database, accessed by API (maximum page size requested was 100).

The subject of this project was inspired by the Field Studies Council WildID guide ‘Using a lichen based index to nitrogen air quality’. The species growth forms used are bushy lichen, leafy lichen and crustose or granular lichen.
